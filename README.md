# Aaron's Dotfiles

## Prerequisits

* [Install Chezmoi](https://www.chezmoi.io/install/)

## Setup

```
chezmoi init https://github.com/AutomateAaron/dotfiles/
```

```
chezmoi apply
```

## Post Setup
1. Install [Oh my zsh](https://ohmyz.sh/) (and plugins)
1. Install [SauceCodePro Nerd Font](https://github.com/ryanoasis/nerd-fonts/releases/tag/v3.4.0)
1. Install [docker](https://docs.docker.com/engine/install/)

See [setup instructions here](https://www.chezmoi.io/quick-start/#using-chezmoi-across-multiple-machines)
